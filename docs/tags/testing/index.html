Unit Testing in Rust (or why I love Rust)

If you’ve been through the process of adding unit tests in C++ with gmock or OCaml or python you know that you at least need to create a new file, add some linking flags, import a library and trigger the test suite from code and then execute the test program sepately. No wonder it takes some painful code review comment to get people to start writing tests. Rust turns that pain into pleasure....
